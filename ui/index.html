Setup

Quick start

Up and running in under 5 minutes.

1. Install

# Clone the repo git clone https://github.com/mattycigemp-crypto/GOD-OPENCODE cd GOD-OPENCODE # Run the one-command installer .\god-install.ps1 # Verify installation .\god-health.ps1

2. Use slash commands

# Build a new application /build a FastAPI REST API with JWT auth, PostgreSQL, and Alembic migrations # Debug a problem /debug users are getting duplicate emails on registration # Security audit /secure review the authentication flow # Architecture review /architect should I use PostgreSQL or MongoDB for a multi-tenant SaaS?

3. Invoke agents directly

# Direct agent invocation As the database-architect, design a schema for a multi-tenant SaaS application where each tenant's data must be isolated. As the security-engineer, audit this JWT implementation: [paste code] As the ai-engineer, design a RAG pipeline for a customer support chatbot with 50k support articles.
