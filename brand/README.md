# GOD-OPENCODE Brand Assets

## Logo Files

| File | Usage | Size |
|------|-------|------|
| `icon.png` | App icon, favicon, GitHub avatar | 1024x1024 |
| `wordmark.png` | README hero, marketing | 1200x400 |
| `full-logo.png` | Complete logo with icon + text | 1200x600 |
| `icon-crowned.png` | Premium/marketing variant | 1024x1024 |

## Colors

| Name | Hex | Usage |
|------|-----|-------|
| Violet | `#8b5cf6` | Primary accent, gradients |
| Cyan | `#06b6d4` | Secondary accent, gradients |
| Background | `#09090b` | Dark theme base |
| Surface | `#18181b` | Cards, panels |
| Text | `#fafafa` | Primary text |

## Usage

- **GitHub avatar**: Use `icon.png`
- **README header**: Use `wordmark.png` or `full-logo.png`
- **Favicon**: Use `icon.png` (16x16 and 32x32 variants)
- **Social sharing**: Use `full-logo.png` with OpenGraph dimensions (1200x630)
